# Digital Portfolio - Priti Dalvi

Welcome to my digital portfolio! This repository contains the source code for my personal student portfolio website, designed to showcase my accomplishments, skills, and projects as an Artificial Intelligence and Machine Learning (AIML) student.

## 🌟 Features
- **Responsive Design**: Flawless viewing experience across devices (Mobile, Tablet, Desktop).
- **Dark Mode Theme**: Sleek, modern aesthetic featuring glassmorphism design elements.
- **Micro-animations**: Smooth scrolling, scroll-reveal transitions, and dynamic typing effects.
- **Masonry Gallery**: Interactive gallery displaying my certifications.
- **Project Showcase**: Detailed cards highlighting my key technical projects.

## 🛠️ Built With
- **HTML5**: Semantic and accessible website structure.
- **CSS3 (Vanilla)**: Advanced styling using CSS Variables, Flexbox, Grid, and Animations.
- **JavaScript**: Interactive logic including a light/dark mode theme toggle and Intersection Observer for scroll animations.
- **FontAwesome**: High-quality icons for social links and UI elements.
- **Google Fonts**: Modern typography (`Outfit` and `Inter`).

## 🚀 How to View
To browse the portfolio on your local machine:
1. Download or clone the repository.
2. Open `index.html` directly in your favorite web browser (Chrome, Edge, Firefox, Brave).
3. Alternatively, you can run a local server (e.g., `python -m http.server` or `npx serve`) in the project directory.
